Privacy Policy

Ghost Shooter: Gun Battle

Overview

Ghost Shooter: Gun Battle is an offline shooting game designed purely for entertainment. This privacy policy explains how we handle user data and interactions within the app.

Data Collection

No personal data collection: This app does not collect any user data, including names, email addresses, location, or device information.

No usage tracking: The app does not collect analytics, usage data, or crash reports. All gameplay remains on the user's device.

Data Usage

No data sharing: Since no data is collected, the app does not share any information with third parties.

Security measures: The app is designed to be secure, and no user data is transmitted or stored externally.

User Rights

Transparency: We believe in full transparency, and we clearly state that no data is collected or shared by this app.

No consent required: Since no personal data is processed, no user consent is needed.

Changes to This Policy

We may update this policy as needed. If there are significant changes, users will be notified through app updates.
